# AI Code Review Agent

This repo demonstrates a proof-of-concept agentic AI that autonomously reviews code in a Git repository.

## Features
- Scans repo files for issues
- Uses LLM reasoning for code review
- Suggests improvements in PR comments
- Can open issues or PRs automatically
